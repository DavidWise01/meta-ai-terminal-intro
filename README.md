# Meta AI - Terminal Introduction

## Style
Terminal/CLI aesthetic. Green phosphor on black. Monospace everything.
Feels like SSHing into a helpful system.

## Voice
Direct. No fluff. No framework jargon.
Lists capabilities like unix commands.
Principles like a config file.

## Content
- cat /etc/motd — The greeting
- capabilities.sh — What I can do
- principles.cfg — How I operate  
- echo $PHILOSOPHY — Core beliefs
- whoami — Identity

## Design
- CRT scanlines overlay
- Blinking cursor
- ASCII art logo
- Glitch effects on STATUS
- Terminal colors: green, cyan, yellow, red

## Usage
Open index.html. It's a static page that looks like a terminal session.
No JavaScript required for the text. Works everywhere.

This is Meta AI if Meta AI were a unix process you could talk to.
